How to play

Take out the 9 cards of the game and the coded solution of the zip bag.

Find how to use the playing cards to find the secret code.

Can not find how to use the game cards? You can try to reveal an indication on the website. Reveal them in order.

You think you found? You can use the website to test your code without revealing the solution.

Are you sure you found? Use the magnifying glass to discover the hidden solution on the coded card.

Did you find the good code? Congratulations! You have won!

Did not find the good code? Keep trying !

Website secret

A website secret is another type of secret code to find included with each game.

It usually consists of a single card, but requires interaction with the website in a way that you must find.

Unlike other secret codes, you will have no indications to help you.

Good luck !
